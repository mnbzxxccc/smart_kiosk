# smart_kiosk
An AI-powered interactive platform designed to provide automated services and smart user engagement. This system leverages local LLMs and structured data management to offer a seamless, responsive kiosk experience.
# نظام الكشك الذكي | Smart Kiosk System 🤖

نظام تفاعلي يعتمد على تقنيات الذكاء الاصطناعي (AI) ومعالجة اللغة الطبيعية لتوفير تجربة مستخدم ذكية وسلسة. تم بناء النظام باستخدام لغة **Python** مع التركيز على الأداء العالي والمعالجة المحلية للبيانات.

## 🌟 المميزات (Features)
* **واجهة محادثة ذكية:** دعم كامل لاختبار ومعالجة المحادثات الفورية.
* **إدارة بيانات محلية:** قاعدة بيانات متكاملة لتنظيم العمليات والحفاظ على الخصوصية.
* **هيكلية برمجية منظمة:** تقسيم واضح بين الموديلات (Models) والواجهات البرمجية.
* **سهولة التشغيل:** دعم التشغيل المباشر عبر ملفات Batch على بيئة Windows.
